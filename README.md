# Learner's High Decision Context

이 레포지토리는 Toss Learner’s High 프로그램에 참여하며,  
기존 Zero Trust POC 백엔드를 직접 사용하고 디버깅해보는 과정에서 시작된 프로젝트입니다.

초기 POC에서는 OPA에서 전달된 정책 평가 로그를 그대로 DB에 저장하고, 이를 조회하는 방식으로 디버깅을 진행하고 있었습니다.  
기본적인 확인에는 문제가 없었지만, 접근 요청이 거절된 경우 “왜 Deny 되었는지”, 그리고 “판단이 의도한 대로 이루어진 것인지”를  
빠르게 파악하기는 쉽지 않았습니다.

이 레포지토리는 이러한 불편함의 근본적인 원인이 무엇인지 고민하고, 이를 어떻게 개선해볼 수 있을지 탐색한 과정을 기록한 프로젝트입니다.
